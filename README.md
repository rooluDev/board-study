# 📋 게시판 사이트

## 📝 프로젝트 개요
이 프로젝트는 게시판의 기본적인 기능(글 작성, 목록, 수정, 삭제, 파일 업로드 등)을 Model1(JSP), Model2(JSP & Servlet), MVC(Spring Boot), SPA(Vue.js & Spring Boot) 4가지 구조로 각각 구현하여, 웹 개발의 핵심 아키텍처와 개발 패턴을 비교·학습할 수 있도록 구성하였습니다.
